# 고스돕 튜터 테이블

이 프로젝트는 표준 고스돕을 기준으로 초보에게 매 턴 튜터링을 해주는 3인 연습판이다.

## 문서 구조
- [기본룰](./docs/gostop-core-rules.md)
- [튜터 규칙](./docs/gostop-tutor-spec.md)
- [UI 규칙](./docs/gostop-ui-spec.md)

## 현재 구현 요약
- 3인 진행, 첫 판 선 랜덤, 승자 다음 판 선
- 조커 2장 기본 포함, UI에서 켜기/끄기 가능
- 시작 바닥 조커는 선이 먼저 가져감
- 총통 10점 승리
- 고/스톱 추천
- 흔들기 선택 및 상대 흔들기 반영
- 쪽/쓸 피 이동, 일반피 우선
- 9월 술잔은 표준룰 기준으로 자동 겸용 처리

세부 구현 기준과 사용자 요청 반영 사항은 위 세 문서를 기준으로 계속 업데이트한다.
