# 고스돕 기본룰

## 기준
- 한국에서 통용되는 표준 고스돕 룰을 우선한다.
- 로컬룰과 옵션룰은 기본룰과 분리해서 본다.

## 카드 분류
- 광, 열끗, 띠, 피 분류를 기본으로 쓴다.
- 비광 포함 3광 2점, 일반 3광 3점, 4광 4점, 5광 15점이다.
- 열끗 5장부터 1점, 띠 5장부터 1점, 피 10장부터 1점이다.
- 정산 금액은 기본 3점(300원)부터 시작하고, 1점 = 100원으로 계산한다. 판이 끝나 승자가 1점이나 2점이어도 수령액은 3점 기준으로 본다.
- 쌍피는 피 2장 가치다.
- 홍단, 청단, 초단은 각 3점이다.
- 고도리는 5점이다.

## 진행
- 고/스톱 선언은 기본 3점부터 가능하다.
- 다만 손패나 더미가 먼저 다하면 패 소진으로 판이 끝나고, 그때는 최고 점수로 승패를 가린다.
- 손패 1장을 내고 같은 월을 가져간다.
- 더미에서 1장을 뒤집어 같은 방식으로 처리한다.
- 같은 월 1장이면 그 카드와 가져간다.
- 같은 월 2장이면 1장을 선택해 가져가고 1장은 남긴다.
- 같은 월 3장이면 4장을 모두 가져간다.

## 특수 규칙
- 총통: 시작 손패에 같은 월 4장이면 10점 승리다.
- 첫 판 선은 랜덤, 승자는 다음 판 선이다.
- 조커 2장은 기본 포함이며, 시작 바닥패에 깔리면 선이 먼저 가져간다.
- 쪽/쓸이 발생하면 상대에게서 피 1장씩 가져오며, 일반피를 우선 가져간다.
- 9월 국화 술잔은 겸용 카드이며, 잡을 때 선택하지 않고 현재 점수상 더 유리한 쪽으로 자동 처리한다.

## 옵션/확장
- 조커는 UI에서 켜기/끄기 가능하다.
- 흔들기, 폭탄, 박, 배수는 현재 프로젝트 규칙으로 따로 관리한다.
