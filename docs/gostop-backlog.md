# 고스돕 전체 이슈 백로그

이 문서는 초창기 요청부터 최근 지적까지, 사용자 피드백을 개발 항목으로 다시 묶은 기준 문서다.
상태 값은 아래처럼 쓴다.

- `완료`: 사용자 요구가 코드와 UI에 반영됨
- `부분 반영`: 일부 반영됐지만 품질이나 일관성이 아직 부족함
- `미완료`: 아직 제대로 반영되지 않음
- `치명`: 플레이 멈춤처럼 우선순위가 가장 높은 항목

## 1. 치명적 진행 버그

### 1-1. 내 턴 뒤 상대 A/B 차례가 안 넘어가고 멈춤
- 상태: `치명`
- 메모:
  - 가장 반복적으로 보고된 문제다.
  - 흔들기, 고/스톱, 일반 턴 모두에서 발생 가능하다.
  - 임시 watchdog, fallback timer는 여러 번 넣었지만 아직 완전 종결되지 않았다.
  - 구조적으로는 `카드 모션 종료 -> 턴 종료 -> 다음 플레이어 확정 -> AI 예약`이 한 번만 실행되게 더 정리해야 한다.

### 1-2. 선택창/모달 때문에 턴이 막히는 문제
- 상태: `부분 반영`
- 메모:
  - 고/스톱은 패를 가져오고 난 뒤에 뜨도록 조정함.
  - 하지만 다른 선택 상태와 턴 종료가 여전히 충돌할 수 있다.

## 현재 최우선 작업
- `치명`: 내 턴 뒤 상대 A/B 차례가 멈추지 않도록 턴 종료와 모션 완료를 한 흐름으로 재정리
