# learning_gostop

간단한 룰은 아는데, 고스돕 실력을 향상하고 싶은 초보자들을 위한 튜터링 웹앱입니다.

## 문서
- `harness/README.md`: 하네스 실행법, 100점 검수 게이트, acceptance/visual/runtime 체크 확장 방법
- `docs/gostop-core-rules.md`: 기본룰, 점수, 고/스톱, 정산 기준
- `docs/gostop-tutor-spec.md`: 추천 로직, 위험 경고, 전략, 종료 해설 기준
- `docs/gostop-ui-spec.md`: 화면 배치, 카드 정리 방식, 모달/패널 UI 기준, 우리끼리 쓰는 UI 용어 규칙
- `docs/gostop-backlog.md`: 전체 이슈 목록, 완료/부분 반영/미완료 상태표
- `research/`: 같은 게임 룰, 유사 UX, 승리 전략, 채택 후보/채택/거절 기록

## 하네스 운영 원칙
- 하네스는 게임 런타임에 싣지 않는다. 모바일 성능과 별개로, 개발 검수 단계에서만 실행한다.
- 하네스는 `구조`, `요구사항`, `리서치/채택`, `승리 전략`, `시각`, `턴 흐름`, `백로그`를 100점 기준으로 검수한다.
- 유사 게임/유사 UX에서 가져올 점은 먼저 `research/`에 기록하고, 실제로 채택했으면 어떤 문서와 코드에 반영했는지까지 남긴다.
