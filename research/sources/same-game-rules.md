# 같은 게임 룰 리서치

## 목적
- 같은 고스톱/맞고 계열에서 어디까지를 `표준룰`, 어디부터를 `옵션룰`로 둘지 선을 분명히 잡는다.
- 점수, 정산, 박 규칙, 조커 규칙처럼 오해가 잦은 부분을 하네스에서 다시 확인할 기준 문서로 쓴다.

## 현재 프로젝트 기준
- `표준룰`: 고/스톱은 3점부터 가능, 비광 포함 3광 2점, 일반 3광 3점, 4광 4점, 5광 15점, 피박/광박 적용.
- `옵션룰`: 조커, 첫뻑, 쪽/쓸 추가 피 이동, 조커 뺏김, 집마다 다른 배수 규칙.
- `고/스톱`: 최소 3점부터 선택 가능하며, 고 횟수는 공개 정보로 보여줘야 한다.
- `광박`: 광으로 이긴 경우 상대 광이 0장이면 배수 적용 여부를 설명해야 한다.
- `피박`: 피로 이긴 경우 상대 피가 적으면 배수 적용 여부를 설명해야 한다.
- `조커`: 현재 프로젝트에서는 옵션룰이며, 켰을 때도 피 1장 가치로만 계산한다.

## 하네스가 보는 최소 키워드
- 표준룰
- 옵션룰
- 조커
- 피박
- 광박
- 고/스톱
